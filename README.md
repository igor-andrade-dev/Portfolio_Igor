## ⚙️ Installation

Clone the repository:

```
git clone https://github.com/IgorDevFullstack/Portifolio_Igor.git
```

Navigate to the project folder:

```
cd Portifolio_Igor
```

Install dependencies:

```
npm install
```

Run the development server:

```
ng serve
```

Open in your browser:

```
http://localhost:4200
```
